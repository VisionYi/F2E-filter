## 專案使用的技術
- Vue、Vuex、vue-router
- SCSS
- Axios
- 串接第三方 API (高雄市政府 Open Data)
- Vuetify 基於 vue2.x 的 UI 組件框架

## 開發專案使用指令
```shell
# 初始化專案
npm install

# 開啟 vue server
npm run server

# 建置發布的資料 dist
npm run build

# 自動佈署到 Github Page 上
npm run deploy
```
